Stool Samples

A set of stools made from CNC cut osb board

Design

Needed some stools / stands so designed a set made from CNC cut OSB boards. The idea was to create a family of figures which are sampled from an infinite set based on primitive shapes in a grid layout.

This enables a creation of a group of cohesive pieces without actual repetition, that can easily be extended when needed.
