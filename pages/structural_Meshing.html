During my time at AKT II, as part of the development of a grasshopper based interoperability toolkit, a side project has been developing tools for FE meshing. Some FE packages, especially sofistik, can generate quality quad meshes but others are a bit unreliable. Driving the model from a core GH definition posed the need to create good meshes for FEA in the grasshopper environment. To do this I have developed various tools, both using external utilities or custom algorithms. Some examples are shown below.
